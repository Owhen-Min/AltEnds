## 0. Commit Rules
---
> 아래의 규칙에 맞춰서 Commit message를 작성합니다.
> commit message : 'type: body / Issue'
> 	- type(타입) : title(제목)
> 	- body(본문, 생략 가능)
> 	- Issue : #issueNo, ...(참고 이슈, 생략 가능) / 생략 시 None

### 1) 예시) feat: navbar 또는 docs: README.md

### 2) Type
| Type 키워드 | 사용 시점                                         |
| -------- | --------------------------------------------- |
| feat     | 새로운 기능 추가                                     |
| fix      | 버그 수정                                         |
| docs     | 문서 수정                                         |
| style    | 코드 스타일 변경 (코드 포매팅, 세미콜론 누락 등)<br>기능 수정이 없는 경우 |
| design   | 사용자 UI 디자인 변경 (CSS 등)                         |
| test     | 테스트 코드, 리팩토링 테스트 코드 추가                        |
| refactor | 코드 리팩토링                                       |
| build    | 빌드 파일 수정                                      |
| ci       | CI 설정 파일 수정                                   |
| perf     | 성능 개선                                         |
| chore    | 빌드 업무 수정, 패키지 매니저 수정 (gitignore 수정 등)         |
| rename   | 파일 혹은 폴더명을 수정만 한 경우                           |
| remove   | 파일을 삭제만 한 경우                                  |
### 3) Issue
| 사용 시점 | 사용 키워드                                |
| ----- | ------------------------------------- |
| 해결    | Closes(종료), Fixes(수정), Resolves(해결)   |
| 참고    | Ref(참고), Related to(관련), See also(참고) |

## 1. 메인 페이지 (HomeView)

- **목적**: 사용자에게 주간 영화 목록과 대체 결말, 사용자 랭킹을 제공
- **구성 요소**:
    - **헤더**
        - 서비스 로고 (클릭 시 메인 페이지로 이동)
        - 네비게이션 바
            - **로그인**, **로그아웃**, **회원가입** 버튼이 우측에 위치
            - 로그인 시에는 **로그아웃** 버튼만 표시
    - **영화 리스트 섹션**
        - 주간 인기 영화 리스트를 카드 형식으로 표시
        - 각 카드에는 영화 이미지, 제목, 간략한 소개가 포함되며 클릭 시 **MovieListDetailView**로 이동
    - **대체 결말 랭킹 섹션**
        - 상위 1, 2, 3위 대체 결말을 카드 형식으로 표시
        - 각 대체 결말 카드에 영화 제목, 결말 텍스트 미리보기, 좋아요 수 표시
        - 클릭 시 **EndingListDetailView**로 이동
    - **사용자 랭킹 섹션**
        - 상위 1, 2, 3위 사용자 카드 표시
        - 사용자명, 총 좋아요 수, 활동 점수 표시
    - **푸터**
        - 서비스 소개, 이용 약관, 개인정보 처리방침 링크

---

## 2. 로그인 페이지 (LoginView)

- **목적**: 사용자 계정 로그인
- **구성 요소**:
    - **로그인 폼**
        - 사용자 아이디와 비밀번호 입력 필드
        - 에러 메시지 표시 영역 (아이디 또는 비밀번호 오류 시)
    - **로그인 버튼**
        - 입력한 정보로 로그인을 시도
        - 로그인 성공 시 메인 페이지로 리다이렉트
    - **회원가입 페이지 이동 버튼**
        - 회원가입 페이지로 이동할 수 있는 버튼

---

## 3. 회원가입 페이지 (SignUpView)

- **목적**: 새로운 사용자 계정 생성
- **구성 요소**:
    - **회원가입 폼**
        - 사용자 아이디, 비밀번호, 비밀번호 확인, 닉네임, 이메일 입력 필드
        - 비밀번호 일치 여부 확인 메시지 표시
    - **회원가입 버튼**
        - 입력한 정보로 계정 생성 시도
        - 가입 성공 시 로그인 페이지로 이동
    - **취소 버튼**
        - 메인 페이지로 돌아가는 버튼

---
## 4. 사용자 프로필 페이지 (ProfileView)

- **목적**: 사용자의 활동 관리 및 정보 설정
- **구성 요소**:
    - **프로필 정보 섹션**
        - 사용자 프로필 사진, 닉네임, 가입 일자 표시
    - **내가 작성한 대체 결말 섹션**
        - 작성한 대체 결말 목록, 좋아요 수, 댓글 수, 잔여 토큰 표시
    - **설정 섹션**
        - 프로필 편집, 로그아웃 버튼

---
## 5. 원본 영화 리스트 페이지 (MovieListView)

- **목적**: 모든 영화 리스트를 제공하고, 영화별 상세 페이지로 이동 가능
- **구성 요소**:
    - **영화 리스트 섹션**
        - 모든 영화를 카드 형식으로 표시
        - 영화 이미지와 제목 표시, 클릭 시 **MovieListDetailView**로 이동
    - **검색 및 필터 기능**
        - 영화 제목 검색 기능 제공
        - 필터 옵션 (예: 장르별, 평점순 등)

---

## 6. 원본 영화 상세 페이지 (MovieListDetailView)

- **목적**: 원래 결말과 대체 결말 생성 페이지로 이동 기능 제공
- **구성 요소**:
    - **영화 기본 정보 섹션**
        - 영화 포스터, 제목, 개요 표시
    - **원래 결말 섹션**
        - 영화의 원래 결말 텍스트 표시
        - 기본 대체 결말을 생성할 수 있는 버튼
    - **대체 결말 생성 버튼**
        - **EndingListCreateView**로 이동하는 버튼

---

## 7. 대체 결말 리스트 페이지 (EndingListView)

- **목적**: 대체 결말에 대한 토론과 사용자 평가 기능 제공
- **구성 요소**:
    - **게시글 리스트 섹션**
        - 카드 형식으로 각 대체 결말 토론 글 정리
        - 카드에는 제목, 작성자, 댓글 수, 좋아요 수 표시
        - 클릭 시 **EndingListDetailView**로 이동
    - **게시글 상세 보기**
        - 특정 대체 결말에 대한 상세 내용, 댓글 목록, 작성 기능 제공
        - 사용자 의견이나 평가(댓글, 좋아요 등)를 표시할 수 있는 공간
    - **대체 결말 정렬/필터 옵션**
        - 인기순, 최신순, 평점순 등의 필터링 옵션 제공

---

## 8. 대체 결말 상세 페이지 (EndingListDetailView)

- **목적**: 특정 대체 결말에 대한 내용과 사용자 의견 확인, 댓글 작성 및 좋아요 기능 제공
- **구성 요소**:
    - **대체 결말 정보 섹션**
        - 영화 제목, 대체 결말 작성자, 작성 날짜, 좋아요 수, 댓글 수 표시
        - 대체 결말 본문 텍스트 표시
    - **댓글 섹션**
        - 댓글 목록 표시
        - 댓글 작성 기능 (로그인한 사용자만 작성 가능)
    - **좋아요 버튼**
        - 대체 결말에 대한 좋아요 버튼 제공
        - 좋아요 수 업데이트 표시

---

## 9. 대체 결말 생성 페이지 (EndingListCreateView)

- **목적**: 대체 결말 생성 기능 제공
- **구성 요소**:
    - **영화 기본 정보 섹션**
        - 영화 포스터, 제목, 개요를 카드 형식으로 제공
    - **파라미터 입력 섹션**
        - 대체 결말을 생성하기 위해 ChatGPT에게 보낼 파라미터 입력 공간 제공
        - 사용자로부터 간단한 텍스트 입력 받음 (예: 특정 장면을 추가하는 요청 등)
    - **생성 및 취소 버튼 섹션**
        - 대체 결말 생성 및 취소 버튼
        - 생성 버튼 클릭 시 대체 결말이 생성되어 **EndingListView**로 리다이렉트
---

## 10. 커뮤니티 게시판 페이지(CommunityView)

- **목적**: 자유로운 소통을 위한 게시판 기능 제공
- **구성 요소**:

        - 사용자들이 작성한 게시글을 카드 형식으로 정리
        - 카드에는 제목, 작성자, 댓글 수, 좋아요 수 표시
    - **게시글 상세 보기**
        - 게시글에 대한 상세 내용, 댓글 목록, 작성 기능 제공
        - 사용자 의견이나 평가(댓글, 좋아요 등)를 표시할 수 있는 공간
    - **게시글 정렬/필터 옵션**
        - 인기순, 최신순, 평점순 등의 필터링 옵션 제공

    - **게시글 내용 섹션**
        - 영화 제목, 내용, 작성자, 작성 날짜, 좋아요 수, 댓글 수 표시
    - **댓글 섹션**
        - 댓글 목록 표시
        - 댓글 작성 기능 (로그인한 사용자만 작성 가능)
    - **좋아요 버튼**
        - 대체 결말에 대한 좋아요 버튼 제공
        - 좋아요 수 업데이트 표시

## 12. 커뮤니티 게시글 작성 페이지(CommunityCreateView)
